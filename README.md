# 🎓 Smart Minds Kindergarten

A modern, responsive website for a kids' school designed to create a joyful and engaging online presence.

🔗 **Live Site:** https://azizulislamch.github.io/Smart-Minds-Kindergarten/  
<br>
💻 **GitHub Repo:** https://github.com/Azizulislamch/Smart-Minds-Kindergarten

---

## 🌟 Overview

**Smart Minds Kindergarten** is a beautifully designed educational website focused on young learners.  
It highlights a fun, creative, and engaging environment where children can learn, grow, and explore.

---

## ✨ Features

- 📱 Fully responsive (Mobile, Tablet, Desktop)
- 🎨 Modern UI with gradients and clean design
- 🧸 Kid-friendly colorful layout
- 📚 Curriculum section with interactive cards
- 📰 Stories/Blog section
- 🎯 Smooth scrolling text animation
- 🔘 Interactive buttons with hover effects
- 🦶 Responsive footer

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- Google Fonts  
- Responsive Design (Media Queries)

---

## 📱 Responsive Design

The website is optimized for:
- 📱 Mobile devices  
- 📲 Tablets  
- 💻 Desktop screens  

---

## 🚀 Getting Started

To run this project locally:

```bash
git clone https://github.com/Azizulislamch/Smart-Minds-Kindergarten.git
```
Then open index.html in your browser.
